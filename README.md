# khmer-vowel-analysis

音声学では、A vowel diagram や vowel chartと呼ばれる母音を配置したものがある。英語のIPA記号を配置した図は以下のように描かれる。

https://en.wikipedia.org/wiki/Vowel_diagram

母音は、発生する時に舌の位置と開き方が変化する。

ɑ=>ɔ=>o=>u

と図の関係は以下のようになる。この時、F1とF2が変化する。
